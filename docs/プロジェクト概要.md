# State Flow Visualizer

リアクティブプログラミング（Angular, React）における状態管理の複雑さを可視化するツール。

## 概要

モダンなフロントエンド開発では、コンポーネント間の状態管理が複雑化しています。このツールは：

- **データフロー**：Store → Component（props/subscribe経由）
- **イベントフロー**：Button.onClick → dispatch(action) → Store更新
- **状態スコープ**：ローカル / サブツリー / グローバル

これらをシーケンス図として可視化し、開発者がアプリケーションの状態変化を理解しやすくします。

## 技術スタック

| カテゴリ       | 技術                        |
| -------------- | --------------------------- |
| フロントエンド | React 19 + TypeScript       |
| ルーティング   | TanStack Router             |
| 状態管理       | Zustand                     |
| 図描画         | React Flow                  |
| UI             | shadcn/ui + Tailwind CSS v4 |
| 永続化         | IndexedDB (Dexie.js)        |
| ビルド         | Vite                        |

## クイックスタート

```bash
# フロントエンド起動
cd front
npm install
npm run dev
```

http://localhost:5173/ でアクセス
