# State Flow Visualizer

リアクティブプログラミング（Angular, React）における状態管理の複雑さを可視化するツールです。

![State Flow Visualizer](docs/screenshot.png)

## 特徴

- 📊 **シーケンス図表示**: コンポーネント間のデータフロー・イベントフローを可視化
- 🎨 **GUIエディタ**: 直感的な操作でダイアグラムを作成
- 📍 **状態スコープ可視化**: ローカル / サブツリー / グローバルを色分け表示
- 📥 **JSON入出力**: 生成AIでJSONを生成し、インポート可能
- 💾 **ローカル保存**: IndexedDBでブラウザに保存

## クイックスタート

```bash
# フロントエンド起動
cd front
npm install
npm run dev
```

ブラウザで http://localhost:5173/ にアクセス

## 技術スタック

| カテゴリ       | 技術                        |
| -------------- | --------------------------- |
| フロントエンド | React 19 + TypeScript       |
| ルーティング   | TanStack Router             |
| 状態管理       | Zustand                     |
| 図描画         | React Flow                  |
| UI             | shadcn/ui + Tailwind CSS v4 |
| 永続化         | IndexedDB (Dexie.js)        |

## 使い方

1. **ダイアグラムを作成**: 一覧画面で「新規作成」
2. **アクターを追加**: コンポーネント、ストア、サービスなど
3. **状態を定義**: スコープ（ローカル/サブツリー/グローバル）を設定
4. **フローを作成**: イベント発火から状態変更までの流れを定義
5. **可視化**: シーケンス図で確認

## スコープ凡例

| スコープ | アイコン | 説明                 | 例               |
| -------- | -------- | -------------------- | ---------------- |

| subtree  | 🌲       | コンポーネントツリー | useContext, DI   |
| global   | 🌍       | アプリ全体           | Zustand, Redux   |
